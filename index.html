LDD Scripts by LavenderDragonDesign

Required One‑Time Setup

All LDD scripts require Tampermonkey and permission to run user scripts. Do this once per browser.

1) Install Tampermonkey

2) Open Extensions (Puzzle icon)

3) Pin Tampermonkey

4) Manage Extensions

5) Enable Allow User Scripts (Required)
