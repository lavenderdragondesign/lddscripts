Required One-Time Setup

Before installing any LDD script, complete the steps below. This only needs to be done once per browser.

1) Install Tampermonkey

Install the Tampermonkey extension for your browser.

2) Open Extensions Menu

Click the puzzle icon in your browser toolbar.

3) Pin Tampermonkey

Pin Tampermonkey so it is always accessible.

4) Open Manage Extensions

Open the browser extensions settings page.

5) Enable “Allow User Scripts”

This setting is required for all userscripts to run.
